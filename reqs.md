### pxslcc

```
brew install ghc cabal-install wget
wget https://github.com/tmoertel/pxsl-tools/archive/master.zip
unzip master.zip 
cd pxsl-tools-master/
cabal install
make
cabal install
# add ~/.cabal/bin to path
```

### Skicka

```

skicka init
skicka ls
# authorize in browser
```
